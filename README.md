# RA事業計画書 2026

エンジニア採用特化型・シェアリングモデルのRA（リクルーティングアドバイザー）事業計画書です。

## 概要

| 項目 | 内容 |
|------|------|
| 計画期間 | 2026年6月〜12月 |
| 対象職種 | エンジニア職（開発・インフラ・データ等） |
| フィー単価 | 40万円〜（固定 or 年収×20%） |
| 稼働企業数 | 最大10社 |
| 月間決定目標 | 5件（12月末時点） |
| 期間累計売上目標 | 940万円 |

## 事業の特徴

- **少数精鋭モデル**：稼働企業を10社に絞り、管理品質と関係深度を維持
- **シェアリング展開**：エージェントマッチ・サーカス等への求人共有で、自社CA稼働ゼロでも決定が生まれる構造
- **フィー2種類に対応**：一律固定フィー（40万円〜）と年収連動フィー（×20%）を企業ごとに管理
- **データ活用**：選考通過・不通過の分析から「通過ペルソナ」を蓄積し他社展開へ活用

## フェーズ

| フェーズ | 期間 | 目標 |
|---------|------|------|
| Phase 1 立ち上げ期 | 6月〜7月 | 稼働企業5社確保・シェアリング初稼働 |
| Phase 2 実績構築期 | 8月〜9月 | 稼働10社・月3件以上決定 |
| Phase 3 スケール期 | 10月〜12月 | 月5件決定・単価50万円達成 |
