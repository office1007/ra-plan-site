# Instagram SNS運用 事業計画書

**策定日：** 2026年6月12日
**運用開始予定：** 2026年7月
**目的：** 人材紹介事業における求職者集客

---

## 1. 事業概要・コンテンツ戦略

### 運用方針

Instagramを主軸とし、**求職者の信頼獲得 → DM流入 → Googleフォーム → 面談申込**のファネルを構築する。公式LINEへの誘導は行わず、Instagram内でのコミュニケーションを完結させてジャンプ率による離脱を防ぐ。

**投稿フォーマットの役割分担**：**リール＝感想・お客様の声 専用**、**フィードカルーセル＝ノウハウ系 専用**と明確に分ける。「すべての投稿」タブでは両方が表示されるが、リールタブを開くと感想コンテンツのみが並び、サービスへの信頼感を一点集中で伝えられる構造にする。

### コンテンツ2軸

| 軸 | 形式 | 効果 | 月間目標 |
|---|---|---|---|
| **①リアルな声** | リール動画（15〜45秒） | 社会的証明・信頼形成 | 2〜3本 |
| **②転職ノウハウ** | フィードカルーセル | 検索流入・フォロワー獲得 | 4〜6本 |

## 2. 集客ファネル全体設計

```
【発見】リール・フィード投稿の視聴
   ↓
【興味】プロフィールへのアクセス
   ↓
【アクション】DM送信
   ↓
【ヒアリング】Googleフォーム送付・回答
   ↓
【商談】日程調整 → 無料面談
```

## 3. DM対応テンプレート

### STEP 1｜DM受信直後の返信

```

◯◯エージェントの[担当者名]です。
まずは転職に関する現状を簡単に教えていただけますか？

↓ こちらのフォームに約1分でご回答いただけます
[Googleフォームリンク]

ご回答後、すぐに最適な日程をご提案します！
```

### STEP 2｜フォーム未回答者へのフォロー（翌日〜2日後）

```
先日はDMありがとうございました！

フォームのご回答、お時間あるときにいただけると嬉しいです。
転職について「まだ迷っている」「情報収集中」という段階でも

[Googleフォームリンク]
```

```

内容を確認しました。
[名前]さんのご状況にあわせてご相談できればと思います。

以下の日程でご都合はいかがでしょうか？
（オンライン・30分程度）

・◯月◯日（◯）◯時〜
・◯月◯日（◯）◯時〜
・◯月◯日（◯）◯時〜

ご希望の日時をお知らせください！
```

## 4. Googleフォーム設計

**フォームタイトル：** 転職相談 ヒアリングフォーム
**所要時間：** 約1分

| # | 設問 | 選択肢 |
|---|---|---|
| 1 | 現在の履歴書はありますか？ | ある（最新版） / あるが古い / ない |
| 2 | 転職の希望時期はいつ頃ですか？ | 3ヶ月以内 / 半年以内 / 1年以内 / まだ決めていない |
| 3 | 転職後の職種について相談したいことはありますか？ | ある（方向性未定） / ある（職種を絞りたい） / ない |
| 4 | InstagramアカウントID（@から始まる） | 記述式 |

### フォーム設定ポイント

- **回答後メッセージ：** 「内容確認後、DMにてご連絡します（通常1営業日以内）」
- **メール通知：** 回答のたびに通知が届くよう設定
- **スプレッドシート連携：** 回答を自動蓄積し顧客管理台帳として活用

## 6. アカウント設計

### BIO文（バイオ文）

プロフィールページの説明文（最大150文字）。**3段構造**で構成する。

```
✅ 転職をトータルサポート
📊 内定率◯% ｜ 累計◯名支援
「迷っている段階」でも相談OK
↓ 転職相談はDMへ（無料）
```

| # | 要素 | 説明 |
|---|---|---|
| ① | 何をするアカウントか | サービス内容を一言で |
| ② | 実績・数字 | 信頼性の根拠 |
| ③ | CTA | DM誘導の一文 |

### リンクインバイオ

Instagramのプロフィールに設置できる唯一のクリッカブルURL。**Linktree（無料）** を使い、1URLから複数の導線に振り分ける。

| ボタン名 | リンク先 |
|---|---|
| 📋 転職相談フォーム | Googleフォーム |
| 📅 面談を予約する | Calendly（拡大期） |
| 🏢 サービス紹介・会社HP | 自社サイト |

### 投稿フォーマット

| 形式 | 役割 | 備考 |
|---|---|---|
| **リール（9:16）** | 感想・お客様の声 専用 | 15〜45秒・テロップ必須。リールタブには感想のみ表示 |
| **フィードカルーセル** | ノウハウ系 専用 | 複数枚スライドで保存率を狙う |
| **ストーリーズ** | DM誘導・リンクスタンプ活用 | 週3〜5回 |

### ハッシュタグ戦略

| 種別 | 個数 | 例 |
|---|---|---|
| ビッグタグ（100万〜） | 2〜3個 | #転職 #キャリアアップ |
| ミドルタグ（10〜100万） | 5〜7個 | #転職活動中 #転職ノウハウ #転職エージェント |
| スモールタグ（1〜10万） | 3〜5個 | #転職エージェントおすすめ #第二新卒転職 |
| ブランドタグ | 1個 | #◯◯キャリア |

### 投稿CTA設計

**リール末尾テロップ**
```
転職の相談はDMへ
↓
簡単なフォームをお送りします
```

**キャプション末尾（固定文）**
```
━━━━━━━━━━━━━━
転職について相談したい方へ

このアカウントにDMを送ってください。
簡単なヒアリングフォームをお送りし、
あなたの状況にあわせてご提案します。

まだ転職するか迷っている段階でもOKです。
━━━━━━━━━━━━━━
```

## 7. KPI・数値目標

> 初期フェーズは**フォロワー数よりDM件数を優先**して観測する。

### フォロワー数目標

| 時期 | 目標フォロワー数 | 重点指標 |
|---|---|---|
| 7月末（1ヶ月） | 30〜80人 | DM月3件 |
| 8月末（2ヶ月） | 80〜200人 | DM月5件以上・面談化率50% |
| 9月末（3ヶ月） | 200〜400人 | DM月10件以上 |
| 12月末（6ヶ月） | 500〜1,000人 | DM月20件・Calendly導入検討 |

### 主要KPI

| 指標 | 目標 | 説明 |
|---|---|---|
| **月間DM件数（最重要）** | **3件〜（7月）** | 面談につながる直接KPI |
| フォーム回答率 | 60%↑ | DM受信→フォーム回答の割合 |
| 面談化率 | 50%↑ | フォーム回答→面談設定の割合 |
| 投稿保存数 | 各投稿5件↑ | ノウハウ系コンテンツの質の指標 |
| エンゲージメント率 | 3〜5% | （いいね+保存+コメント）÷リーチ |
